Passo 1: Criando o Projeto Vue.js com Vite

Navegue até o diretório que você criou:


cd seudiretório

Crie o projeto Vue.js usando Vite: Execute o comando para iniciar o projeto com o Vite:


npm create vite
  • Nome do Projeto: Será solicitado um nome para o projeto. Você pode nomeá-lo conforme preferir, por exemplo: gerenciamento-salas.
  • Selecione a Framework: Selecione Vue como o framework.
  • Selecione a variante: Escolha Vue (sem TypeScript por enquanto, mas você pode optar por Vue + TypeScript se quiser usar TS).
  • Instale as dependências: Após a criação do projeto, entre na pasta do projeto e instale as dependências:

cd gerenciamento-salas
                

npm install

Execute o projeto:


npm run dev

Isso deve abrir a aplicação Vue.js padrão no navegador.

Passo 2: Organizando a Estrutura de Pastas

Com o projeto criado, vamos organizar as pastas para seu sistema de gerenciamento de salas. Aqui está uma sugestão para organizar as páginas e componentes:

  • Estrutura Inicial de Diretórios: Abaixo está um esboço de como você pode estruturar o projeto de acordo com as funcionalidades mencionadas.

gerenciamento-salas/
├── public/
├── src/
│   ├── assets/
│   ├── components/
│   │   ├── form/
│   │   │   ├── LoginForm.vue
│   │   │   ├── RegisterForm.vue
│   │   └── common/
│   │       ├── Navbar.vue
│   │       ├── Sidebar.vue
│   ├── pages/
│   │   ├── Home.vue
│   │   ├── Login.vue
│   │   ├── Register.vue
│   │   ├── VisualizarSala.vue
│   │   ├── VisualizarLaboratorio.vue
│   │   ├── AdicionarSala.vue
│   │   ├── AdicionarLaboratorio.vue
│   │   ├── AdicionarRecurso.vue
│   ├── router/
│   │   └── index.js
│   ├── store/
│   │   └── index.js
│   ├── App.vue
│   └── main.js
├── package.json
└── vite.config.js
  • Explicação das Pastas:
    • components/: Contém componentes reutilizáveis. Por exemplo, o Navbar.vue para o menu de navegação, o Sidebar.vue para a barra lateral e componentes de formulário (como login e registro).
    • pages/: Contém as páginas principais do projeto, cada uma representando uma funcionalidade (login, registro, visualização e adição de salas e laboratórios).
    • router/: Contém a configuração de rotas para navegar entre as páginas do sistema.
    • store/: Contém o estado global da aplicação (se usar Vuex ou Pinia para gerenciamento de estado).

Passo 3: Configurando o Router

Para gerenciar as rotas entre as páginas (como login, home, etc.), precisamos configurar o vue-router.

  • Instalar o vue-router: No diretório do projeto, rode:

npm install vue-router
  • Configurar o router: Crie o arquivo src/router/index.js com a seguinte configuração básica:

import { createRouter, createWebHistory } from 'vue-router';
import Home from '../pages/Home.vue';
import Login from '../pages/Login.vue';
import Register from '../pages/Register.vue';
import VisualizarSala from '../pages/VisualizarSala.vue';
import VisualizarLaboratorio from '../pages/VisualizarLaboratorio.vue';
import AdicionarSala from '../pages/AdicionarSala.vue';
import AdicionarLaboratorio from '../pages/AdicionarLaboratorio.vue';
import AdicionarRecurso from '../pages/AdicionarRecurso.vue';

const routes = [
  { path: '/', name: 'Home', component: Home },
  { path: '/login', name: 'Login', component: Login },
  { path: '/register', name: 'Register', component: Register },
  { path: '/visualizar-sala', name: 'VisualizarSala', component: VisualizarSala },
  { path: '/visualizar-laboratorio', name: 'VisualizarLaboratorio', component: VisualizarLaboratorio },
  { path: '/adicionar-sala', name: 'AdicionarSala', component: AdicionarSala },
  { path: '/adicionar-laboratorio', name: 'AdicionarLaboratorio', component: AdicionarLaboratorio },
  { path: '/adicionar-recurso', name: 'AdicionarRecurso', component: AdicionarRecurso },
];

const router = createRouter({
  history: createWebHistory(),
  routes,
});

export default router;
  • Atualizar o main.js para usar o router: No arquivo src/main.js, importe o router e configure a aplicação para usá-lo:

import { createApp } from 'vue';
import App from './App.vue';
import router from './router';

createApp(App)
  .use(router)
  .mount('#app');
